# GalaxyBlitz

GalaxyBlitz is a simple neon style space shooter game made by me
this is simple and total fun

this is a game where you willl have your own ship
and you have to shoot the enemies ship to prevent yourself from dyinnng

## Features

- Player spaceship movement which can be controlled through right left arrow keys
- Enemy spawning system ( its infinite )
- the much you shoot the much you get the score
- once the enemy ship got touched with you... GAME OVER!!
- Restart game button

## Controls

- Arrow Left and  Right wich will Move spaceship
- Spacebar will Shoot bullets

## Lang I used

- HTML
- CSS
- Java

## Project Purpose

I made this project to improve my Java  skills by creating a small arcade game instead of a normal white BG website
I wanted to learn how game movement works and most important... the collison effect one
